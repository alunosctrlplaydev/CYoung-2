Links em HTML

Voltar para a página principal

Para criar links, usamos a tag <a> com os atributos:

  • href -- endereço (URL) do link
  • target -- onde abrir o link (_blank = nova janela, _self = mesma janela)

Exemplos de links:

  • Google (link externo)
  • Outra página local
  • Download de arquivo
